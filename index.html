❓ 사용 방법

    

      
🔑 처음 들어가기 (공통)

      
    
        
  1. 주소 접속 후 공용 비밀번호 + 본인 이름을 입력해 들어갑니다.
    2. 
        
  2. 들어가면 [📥 자료 올리기] / [✅ 자막 담당자] 중 하나를 고릅니다.
    3. 
      

      
📱 휴대폰은 브라우저 메뉴에서 “홈 화면에 추가” 해두면 앱처럼 빠르게 열려요.


      자료 올리는 분
      
📥 자료 올리기 (4단계)

      
    
        
  1. ① 날짜 — 자막을 사용할 예배 날짜를 고릅니다.
    2. 
        
  2. ② 예배 — 주일 1부~4부 · 주일오후 · 수요/금요기도회 · 특별집회 · 기타 중 선택.
    3. 
        
  3. ③ 항목 — 📖성경구절 / ✍️자막 / 🎵찬양 콘티 / 🎼찬양 가사 / 🎧기타 음원·링크 중 선택.
    4. 
        
  4. ④ 입력
          
      
            
    • 📖 성경구절: 장·절만 (예: 시편 23:1-4)
      • 
            
    • ✍️ 자막: 띄울 내용을 입력 (빈 줄 = 슬라이드 분할)
      • 
            
    • 🎵 찬양 콘티: 진행·순서 설명을 입력
      • 
            
    • 🎼 찬양 가사: 가사 — 절 사이는 빈 줄로 띄우면 슬라이드 자동 분할 (저장고 불러오기 가능)
      • 
            
    • 🎧 기타 음원·링크: 음원 파일 첨부 + 링크 주소
      • 
          
    
        
    5. 
        
  5. 필요하면 📎 파일 첨부·📝 설명을 더하고 ＋ 등록하기. 등록하면 다시 항목 선택으로 돌아가 계속 올릴 수 있어요.
    6. 
      


      방송실 자막 담당
      
✅ 검수 & 내보내기

      
    
        
  1. ✅ 검수 보드 — 위에서 예배를 고르면 자료가 순서대로 정렬. 📎첨부·💬설명 표시, 누락 자동 알림, 실시간 갱신
    2. 
        
  2. “확인하기” → 미리보기(슬라이드·콘티·음원·설명) 확인 후 ✓ 검수완료
    3. 
        
  3. ⬇️ ProPresenter용 전체 내보내기(.txt) → ProPresenter [텍스트 가져오기]로 불러오면 빈 줄마다 새 슬라이드 생성
    4. 
        
  4. 📖 성경은 ProPresenter 내장 개역개정에서 해당 장·절을 불러오면 됩니다.
    5.
